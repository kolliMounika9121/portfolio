FloodGuard: Safeguarding Communities Against Flooding

Languages Used & How to Install

🧑‍💻 Languages & Frameworks:

  • 📱 Flutter & Dart – for cross-platform mobile app development
  • 🔗 API – OpenWeatherMap API for weather and flood data
  • 📦 JSON – to parse real-time responses
  • 📍 Geolocation & HTTP packages – for live data handling

📲 How to Install the App on Android:

  1. Enable Developer Mode:
    • Go to Settings > About Phone
    • Tap Build Number multiple times until Developer Mode is enabled
    • Go to Developer Options and enable USB Debugging
  2. Connect & Install:
    • Connect your Android phone to your computer via USB
    • Run flutter devices in your terminal to detect the phone
    • Use flutter install to build and install the app on the device

Project Overview

FloodGuard is a community-focused mobile app designed to enhance flood preparedness and safety in rural villages. Developed during a Community Service Project by students of Seshadri Rao Gudlavalleru Engineering College, the app provides:

  • ⏱ Real-time flood risk alerts
  • 🌦️ Weather forecasts and live updates
  • 📘 Flood safety education and emergency guidelines
  • 📍 Location-based flood prediction using OpenWeatherMap API

Objectives

  • ✅ Predict flood risks using real-time weather data (rainfall, temperature, water levels)
  • ✅ Educate communities about flood awareness, prevention and preparedness
  • ✅ Build a user-friendly mobile app that communicates risks and safety practices

App Structure

The FloodGuard app contains the following core sections:

  • 1️⃣ Home Page: Simple UI with navigation to detection, precaution, and prevention
  • 2️⃣ Weather Forecast: Displays next 12-hour weather predictions
  • 3️⃣ Current Weather: Shows temperature, humidity, and condition (uses live API)
  • 4️⃣ Detection Page: Allows users to input location and pincode to get flood risk status
  • 5️⃣ Prevention & Precaution: Tips and safety guidelines for before, during, and after floods

Technologies Used

  • 📱 Flutter & Dart – for cross-platform mobile development
  • 🛰️ OpenWeatherMap API – to fetch real-time weather and flood data
  • 📍 Geolocation – to detect and analyze user’s exact location
  • 🔄 HTTP & JSON – for dynamic API calls and response handling

Key Outcomes

  • 🎯 Delivered a functioning mobile app with real-time detection and alerts
  • 📈 Increased community awareness through in-person workshops
  • 🧠 Developed technical skills in disaster management and app deployment
  • 🛠️ Addressed local flood challenges with tech-enabled solutions
